# Panduan operator MCP

Menghubungkan klien AI ke GoSite lewat plugin `gosite/mcp`.

**Status:** Desain — gelombang P6

## Alur

Install → enable → generate token + pilih scope → salin `gs_pat_*` ke `mcp.json` → spawn MCP stdio di klien AI.

Contoh `mcp.json` dan troubleshooting: [mcp-operator.md](./mcp-operator.md) (EN).
